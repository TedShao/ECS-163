Which state has the largest population?

Hongyu Zhai   -   01/20/2018

Reference

I found this dataset on Kaggle.com. This dataset is contributed by users of Kaggle and it is originally from the US Census Bureau. Link to dataset : https://www.kaggle.com/muonneutrino/us-census-demographic-data

Description

I made a map to show the geographic shape of each state and fill in different color based on the population of that state. In this case, the mark is the shape of a state, and the channel is the color filled in each shape. The color indicates the size of population: deeper color means larger population.

The dataset I am using contain detailed demographic data, so I think a map will be the best way to visualize the data. When visualizing wildfire records or other things that are related to trouble(polution, natural disaster, etc), we tend to use the red color, but it is obviously not appropriate here. In this case, I choose the blue color because it is more netural.

I think my visualization met the requirement, and it is successful in delivering the information to the viewer.
